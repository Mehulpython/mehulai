# mehul.ai

Personal portfolio of Mehul — Software Engineer at Vanguard & AI Builder.

## Tech Stack

- **Framework:** Next.js 16 (static export)
- **Language:** TypeScript
- **Styling:** Tailwind CSS 4 + CSS custom properties
- **Animation:** Framer Motion
- **Icons:** Lucide React
- **Fonts:** Lora (headings) + Raleway (body)

## Features

- Mountain parallax hero with typing animation
- Dark mode toggle with system preference detection
- Interactive project filters (Live / Building / Concept)
- Animated stats counters
- Experience timeline
- Contact form (FormSubmit.co)
- Responsive navbar with mobile menu
- Scroll progress indicator
- View source floating button
- Smooth scroll navigation

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

Static output goes to `out/` — deploy to Vercel, Netlify, or any static host.

## Deploy

Push to `main` — deploy via Vercel or GitHub Pages.
